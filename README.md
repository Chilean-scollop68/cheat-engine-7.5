## Overview
**Cheat Engine 7.5** is the latest version of the most popular open-source memory editor and debugging tool for Windows.

Used by millions of gamers and reverse engineers worldwide to analyze games, create trainers, modify values in memory, and debug applications.

<img width="972" height="744" alt="image" src="https://github.com/user-attachments/assets/e43fb3bb-116d-4653-bb9c-fba358a4bbaf" />

> [!Important]
> Primarily intended for **single-player** and **offline** use.

## Key Features
- Advanced Memory Scanner (Exact Value, Unknown Initial Value, etc.)
- Pointer Scanner & Pointer Map
- Speedhack
- Lua Scripting Engine
- Disassembler & Debugger
- Auto Assembler
- Code Injection
- AOB (Array of Bytes) Scanner
- Trainer Maker
- Built-in Cheat Table support (.CT files)
- Multi-process support

## How to Use
1. Download and extract the latest release
2. Run `Cheat Engine.exe` as Administrator
3. Select the target process
4. Scan for values and create your cheats
5. Save as a Cheat Table for later use

## Supported Games
Works with almost any PC game (Steam, Epic, Origin, etc.) including:
- Single-player story games
- Offline modes
- Emulators
- Many multiplayer games (at your own risk)
